# credit-system-ui
Credit System front end implementation with React.js for Paycore

## Used Technology and Libraries
* React
* Redux (Global State Management)
* React Router (Dynamic Page Navigation)
* Bootstrap 
* Axios (Communication with Backend app)
* React Hooks
